NephroScan
Repository: github.com/ri5hii/nephroscan

AI-assisted kidney disease risk analysis workflow

NephroScan is a clinical decision-support project that helps surface kidney risk signals from routine health data.

It combines structured data processing, risk-model inference, and clear output summaries to support early triage workflows.

Design goal:
- keep clinician-facing output interpretable
- reduce analysis friction
- preserve auditability of each inference run

Core Features

  • Risk score generation from patient features
  • Interpretable factor summary for each prediction
  • Batch analysis mode for screening pipelines
  • Exportable result snapshots for review

Tech Stack

  • Python
  • FastAPI
  • Pandas + NumPy
  • Scikit-learn

Model Pipeline

  • Input validation and feature normalization
  • Model inference on structured patient vectors
  • Post-processing for threshold bands
  • Human-readable risk explanation output

Versioning

  • v0.1 baseline data + training pipeline
  • v0.2 api inference endpoint + reports
  • v0.3 explainability and batch mode
