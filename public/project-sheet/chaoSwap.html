ChaoSwap
Repository: github.com/ri5hii/chaoSwap

Token swap and routing prototype for learning DEX mechanics

ChaoSwap explores AMM-based trade execution, routing behavior, and transaction safety checks in a lightweight experimental stack.

The project focuses on correctness of quote generation and predictable swap flow rather than production exchange volume.

Design goal:
+- transparent swap math
+- deterministic routing behavior
+- clean developer-facing transaction flow

Core Features

  • Pair discovery and pool metadata lookup
  • Quote estimation with slippage controls
  • Swap execution pipeline with status tracking
  • Basic route selection across supported pools

Tech Stack

  • TypeScript
  • Ethers.js
  • Node.js
  • React (dashboard prototype)

Swap Pipeline

  • Input token validation
  • Best-route quote calculation
  • Allowance checks and approval gating
  • Transaction submission + confirmation tracking

Versioning

  • v0.1 pool + quote explorer
  • v0.2 swap execution + status states
  • v0.3 route refinement + UX cleanup
